## 🎯 Milestone Checkpoints

### ✅ Checkpoint 1 (Day 21)
- [ ] `ls`, `cd`, `pwd`, `mkdir` confidently use kar sako
- [ ] Files create, copy, move, delete kar sako  
- [ ] nano editor mein file edit kar sako
- [ ] find command se file dhundh sako

### ✅ Checkpoint 2 (Day 42)
- [ ] File permissions samajh aao (rwxr-xr-x)
- [ ] `chmod` aur `chown` use kar sako
- [ ] Processes list aur kill kar sako
- [ ] systemctl se service start/stop kar sako
- [ ] Log files read kar sako

### ✅ Checkpoint 3 (Day 55)
- [ ] SSH se remote server connect kar sako
- [ ] SSH keys generate aur use kar sako
- [ ] `curl` se API call kar sako
- [ ] SCP se file transfer kar sako

### ✅ Checkpoint 4 (Day 70)
- [ ] Python virtual environment create aur use kar sako
- [ ] Flask app Linux pe run kar sako
- [ ] Environment variables set kar sako
- [ ] Git se code push kar sako

### ✅ Checkpoint 5 (Day 80)
- [ ] Nginx install aur configure kar sako
- [ ] Reverse proxy setup kar sako
- [ ] SSL certificate install kar sako
- [ ] Flask app production mein deploy kar sako

### ✅ Checkpoint 6 (Day 90) — LINUX PRO!
- [ ] Docker image build aur run kar sako
- [ ] Docker Compose se multi-container app chalao
- [ ] Cloud server (Azure/AWS) pe app deploy kar sako
- [ ] Complete production environment manage kar sako
