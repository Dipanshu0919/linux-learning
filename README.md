## 🚀 GitHub Codespaces Mein Kaise Open Karein?

### Step 1: Repository Fork/Clone karein
```bash
# Option 1: GitHub par fork karein (recommended)
# Apne GitHub account mein fork karein

# Option 2: Clone karein
git clone https://github.com/YOUR_USERNAME/linux-learning.git
```

### Step 2: Codespaces Open Karein
1. GitHub pe apna repository open karo
2. Green `<> Code` button pe click karo
3. `Codespaces` tab select karo
4. `Create codespace on main` pe click karo
5. Wait karo — environment ready hoga!

### Step 3: Layout Setup Karein
```
┌─────────────────┬──────────────────┐
│   FILE EXPLORER │   TERMINAL       │
│                 │                  │
│   📄 README.md  │  $ ls -la       │
│   📁 01-Basics/ │  $ pwd          │
│   📁 02-Files/  │  $ echo "Hello" │
│                 │                  │
│   [Markdown     │  [Practice       │
│    Preview]     │   Commands]      │

```
